CURS ÎN DEZVOLTARE WEB

INTRODUCERE

Designul web a devenit un domeniu foarte dezvoltat în primul rând datorită internetului care a redefinit graniţele lumii moderne, astfel încât astăzi fiecare companie mică sau mare, chiar şi o afacere de familie are o prezentare web oficială. Noţiunea de design web presupune un set de diferite competenţe, standarde şi discipline folosite în realizarea paginilor web.

DESPRE

În acest curs veţi cunoaşte dezvoltarea pe scurt a internetului şi a site-urilor, dar veţi obţine şi informaţii detaliate despre principalele diferenţe dintre print şi design web. Pe lângă aceasta, cursul descrie şi elementele designului web, precum şi aplicarea lor pe un site concret.

APLICĂ

După dezvoltarea noilor cunoştinţe, veţi combina toate elementele menţionate şi, respectând bazele unui design bun, veţi fi pregătiţi să creaţi independent pagina web care îndeplineşte cele mai ridicate criterii.
